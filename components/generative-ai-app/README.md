# Crystallize - Generate Content with AI APP

## Requirements

-   Volta.sh (that will bring good version of Node )
-   Docker
-   Node: 20.x
-   PNPM: 9+

## Installation

```bash
make install
```

> Important!: You need to adapt variables in the `.env`.

## Run the project

```bash
make serve
```

## Contributing

-   `make codeclean` will keep the project in order.
